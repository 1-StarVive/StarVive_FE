# 💻 환경설정

다음의 `.env`파일이 필요합니다.

**.env.development**

```
NEXT_PUBLIC_API_URL=https://starvive.store
```

**.env.production**

```
NEXT_PUBLIC_API_URL=https://starvive.store
```

# 👩‍💻 실행

```bash
pnpm i # 설치

pnpm run dev # 개발모드로 실행

# 빌드 후 실행
pnpm run build # 빌드
pnpm run start # 실행
```
